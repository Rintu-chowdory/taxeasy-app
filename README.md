# Vite React Portfolio

## Setup Instructions

Follow these steps to set up your Vite React portfolio:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rintu-chowdory/devops-portfolio-ci-cd.git
   cd devops-portfolio-ci-cd
   ```

2. **Install dependencies**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the development server**
   Start the Vite development server using:
   ```bash
   npm run dev
   ```
   This will start the server and you can view your portfolio at `http://localhost:5173`.

4. **Build for production**
   To create a production build, use:
   ```bash
   npm run build
   ```
   The build files will be generated in the `dist` directory.

5. **Preview the production build**
   You can preview your production build locally with:
   ```bash
   npm run serve
   ```
   This will serve your files from the `dist` directory.

## Notes
- Ensure to have the latest version of Node.js.
- For additional configurations, check the Vite documentation.
